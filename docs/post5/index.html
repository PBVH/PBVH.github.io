Encargo 2.2

Tutoriales

En esta ocasión hice diferentes tipos de moldes siguiendo los tutoriales de webcursos. En esta sección de mi sitio web podrán encontrar el proceso que fui siguiendo.

Tutorial 1

Corte

Para este molde comencé haciendo un sketch como el siguiente, sin medidas definidas ya que las fui modifiando en el proceso.

A continuación utilicé el comando “revolve” para hacer un solido como se muestra en la foto.

Luego modifiqué algunas de las medidas del molde y le di un ángulo de corte a la parte inferior, con el comando “champfer”.

Finalmente afinando algunos detalles, así queda.

Resultado

Tutorial 2

Molde

Al igual que en el anterior comencé haciendo un sketch como el siguiente, en base a un circulo de 60 mm, una curva con puntos aleatorios y una curva cerrada en el plano xy.

Luego utilicé el comando “sweep” para darle volumen a la figura dibujada en el sketch.

En base al mismo circulo del primer sketch realizo una extrusión de este y resto las masas de ambos volumenes con el comando “combine”.

Finalmente con el fin de facilitar el desmoldado se suavizaron los bordes de la figura con el comando “fillet” y se analizó la geometría del objeto con un analisis de acesibilidad y otro de corte, respectivamente.

Resultado

Tutorial 3

Prensa

A diferencia del proceso anterior, este comienza con la creación de un plano y su modificación sobre algunas lineas para que quede de la siguiente manera.

Luego hice un volumen a partir de un cuadrado dibujado un poco más pequeño que el plano anterior.

Después con el comando “thicken” le dí un espesor al plano principal para cortar el volumen del paso anterior con esta figura.

A continuación se hace una especie de caja que contenga el modelo anterior, pasa dar forma a las dos piezas del molde: la tapa y la caja (que se muestra en la imagen).

Finalmente agregué un mango y a través del comando “draft” se inclinaron las caras en base a cierto ángulo, con el fin de poder desmoldar. Y se termina viendo de la siguiente forma.

Resultado

Ejercicio Cura

Una vez instalado el software abrir el archivo que se quiere imprimir desde Fusion 360, en mi caso. Para familiarizarme con el programa inicié cargando una figura sencilla, Luego simulé la impresion con el comando “Slice” y la configuraciones que venían por defecto, y finalmente modifiqué algunas medidas como la densidad del relleno y la forma de este.

Luego, para seguir practicando utilicé uno de los moldes hechos en el turorial 2. Al abrirlo tuve que rotarlo y moverlo para centrarlo.

Finalmente jugué un poco más con la densidad y figura del relleno, y este fue el resultado.
